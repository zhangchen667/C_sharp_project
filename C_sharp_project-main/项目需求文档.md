# **软件工程大作业 \- “我的个人空间” 需求文档（技术强化版）**

## **1\. 项目内容**

本项目为基于 **C\# ASP.NET Core WebAPI** 架构开发的个人综合空间系统。系统采用面向对象（OOP）设计思想，通过前、后端分离的方式实现个人博客、相册存储、访客留言及后台管理功能。项目旨在通过 4 周的敏捷开发，实践软件工程全生命周期管理，并深度集成数据库、异步编程、Linq 查询等核心技术。

## **2\. 系统整体功能**

* **用户与安全模块：**  
  * 利用**正则表达式**实现注册信息的严格校验（如邮箱格式、强密码逻辑）。  
  * 站长登录验证，确保后台管理权限安全。  
* **个人博文模块：**  
  * 支持文章的发布、分类及标签管理。  
  * 集成**Linq**技术实现文章的关键词检索、分类过滤及时间倒序排列。  
* **多媒体相册模块：**  
  * 利用**文件操作**技术实现照片的物理上传与存储。  
  * 前台瀑布流展示，支持图片的查看与删除。  
* **互动留言模块：**  
  * 访客留言功能，后端通过**异步编程**处理高并发写入，保证页面不卡顿。  
* **后台管理与监控：**  
  * 全系统集成全局**异常处理**，记录操作日志，确保系统在高负载或非法输入下的稳定性。

## **3\. 技术实现（对应考核指标）**

为满足大作业选型要求，本项目重点实现以下技术点：

| 考核项类型 | 对应技术点 | 在本项目中的具体应用 |
| :---- | :---- | :---- |
| **必选项** | **面向对象** | 定义 User、Post、Photo 等实体类，封装业务逻辑，利用继承和多态处理不同类型的回复。 |
| **必选项** | **WebAPI** | 采用 ASP.NET Core 编写 RESTful 风格接口，负责与前端数据交互。 |
| **必选项** | **异常处理** | 在 Controller 层设置 try-catch 块，并配置全局异常过滤器，捕获数据库连接错误或 IO 异常。 |
| **必选项** | **数据库** | 使用 SQL Server/MySQL 存储用户数据与文章内容，通过 EF Core 进行对象关系映射。 |
| **可选项 1** | **Linq** | 用于数据库查询优化，如：\_db.Articles.Where(a \=\> a.IsPublic).OrderByDescending(a \=\> a.Date)。 |
| **可选项 2** | **文件操作** | 实现 IFormFile 接口接收上传的照片，并使用 FileStream 将其持久化到服务器硬盘。 |
| **可选项 3** | **正则表达式** | 编写 Regex 表达式校验注册邮箱、手机号及博文中的 HTML 标签过滤。 |
| **可选项 4** | **异步编程** | 所有数据库读写操作均采用 async/await 关键字，提升 Web 服务器并发处理能力。 |

## **4\. 主要业务流程**

1. **发布文章流程：** 站长通过前端提交 JSON 数据 \-\> **WebAPI** 接收请求 \-\> **正则表达式**校验格式 \-\> **异步**写入**数据库** \-\> **Linq** 更新首页列表。  
2. **相册管理流程：** 站长选择图片 \-\> 后端通过**文件操作**保存至服务器路径 \-\> 数据库记录路径 \-\> **异常处理**机制确保上传失败时文件不残留。  
3. **访客留言流程：** 访客提交表单 \-\> 后端捕获并**异步**保存 \-\> 站长后台通过 **Linq** 筛选出未读留言。

## **5\. 人员分工（4人团队）**

* **成员 A（架构师 & 后端）：** 负责**面向对象**实体建模、**数据库**设计及 EF Core 迁移配置。  
* **成员 B（开发核心 & API）：** 负责 **WebAPI** 路由设计、**Linq** 数据检索逻辑及**异步编程**实现。  
* **成员 C（前端 & 安全）：** 负责 UI 界面开发、利用**正则表达式**进行前端验证、对接后端接口。  
* **成员 D（测试 & 运维）：** 负责**异常处理**逻辑测试、**文件操作**（上传下载）的功能校验、编写测试文档及 PPT。

## **6\. 时间进度安排（4周）**

* **第1周：** 需求定稿，完成**面向对象**建模与**数据库** E-R 图设计。  
* **第2周：** 搭建 **WebAPI** 基础框架，完成用户与文章模块的**异步**接口开发，集成 **Linq** 查询。  
* **第3周：** 开发相册**文件操作**功能，完成**正则表达式**校验逻辑，进行全系统的**异常处理**压力测试。  
* **第4周：** 前后端联调，部署项目，汇总文档并准备期末答辩。
