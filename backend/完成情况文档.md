# **"我的个人空间" 项目完成情况文档**

> 最后更新：2026-05-09

---

## 一、已完成功能

### 1. 用户与安全模块

| 功能 | 实现方式 | 涉及文件 |
| :--- | :--- | :--- |
| 用户注册 | ASP.NET Core Identity，支持邮箱、昵称、密码注册，中文错误提示 | Pages/Identity/Account/Register.cshtml(.cs) |
| 用户登录 | 邮箱 + 密码登录，支持"记住我" | Pages/Identity/Account/Login.cshtml(.cs) |
| 用户登出 | POST 方式安全登出 | Pages/Identity/Account/Logout.cshtml(.cs) |
| 角色权限控制 | Admin 角色种子数据，`[Authorize(Roles="Admin")]` 限制管理页面 | Data/SeedData.cs |
| 密码策略 | 必须包含大小写字母、数字、特殊字符，最少 6 位 | Program.cs |

### 2. 个人博文模块

| 功能 | 实现方式 | 涉及文件 |
| :--- | :--- | :--- |
| 文章发布 | 管理员可创建文章，选择分类，设置是否公开，正则过滤 `<script>` 标签 | Pages/Posts/Create.cshtml(.cs) |
| 文章列表 | 展示公开文章，支持 **Linq** 按标题/内容搜索、按分类筛选 | Pages/Posts/Index.cshtml(.cs) |
| 文章详情 | 单篇文章展示，内容换行处理 | Pages/Posts/Details.cshtml(.cs) |
| 分类管理 | 种子数据生成 3 个默认分类，用于文章归类和筛选 | Data/ApplicationDbContext.cs |

### 3. 多媒体相册模块

| 功能 | 实现方式 | 涉及文件 |
| :--- | :--- | :--- |
| 照片上传 | 管理员上传，校验格式（jpg/png/gif/webp）和大小（10MB），保存到 wwwroot/uploads/ | Pages/Albums/Upload.cshtml(.cs) |
| 照片展示 | Bootstrap 网格布局展示所有照片 | Pages/Albums/Index.cshtml(.cs) |
| 照片删除 | 管理员确认删除，同时删除磁盘文件和数据库记录 | Pages/Albums/Delete.cshtml(.cs) |
| 相册模型 | 支持多相册分类，种子数据生成 1 个默认相册 | Models/Album.cs |

### 4. 互动留言模块

| 功能 | 实现方式 | 涉及文件 |
| :--- | :--- | :--- |
| 访客留言 | 提交昵称、邮箱、内容，**正则表达式**校验邮箱格式，**异步编程**保存 | Pages/Comments/Index.cshtml(.cs) |
| 留言展示 | 展示已审核留言及管理员回复 | Pages/Comments/Index.cshtml(.cs) |
| 管理员回复 | 管理员可内联回复留言 | Pages/Admin/ManageComments.cshtml(.cs) |
| 留言删除 | 管理员可删除留言 | Pages/Admin/ManageComments.cshtml(.cs) |

### 5. 后台管理模块

| 功能 | 实现方式 | 涉及文件 |
| :--- | :--- | :--- |
| 管理仪表盘 | 显示文章数、照片数、留言数、待回复数 | Pages/Admin/Index.cshtml(.cs) |
| 评论管理 | 查看所有留言，回复和删除 | Pages/Admin/ManageComments.cshtml(.cs) |

### 6. 考核技术点覆盖情况

| 考核项 | 是否已实现 | 具体应用位置 |
| :--- | :--- | :--- |
| **面向对象**（必选） | ✅ 已实现 | Models/ 下 User、Post、Category、Comment、Photo、Album 实体类，继承 IdentityUser |
| **WebAPI**（必选） | ✅ 已实现 | ASP.NET Core Razor Pages 架构，前后端数据交互 |
| **异常处理**（必选） | ⚠️ 部分实现 | 仅有默认 `/Error` 页面，缺少自定义全局异常中间件 |
| **数据库**（必选） | ✅ 已实现 | MySQL + EF Core (Pomelo)，含迁移文件 |
| **Linq**（可选） | ✅ 已实现 | 文章搜索 `Where(a => a.Title.Contains(...))`、分类筛选、排序 |
| **文件操作**（可选） | ✅ 已实现 | 照片上传 IFormFile + FileStream 持久化到磁盘 |
| **正则表达式**（可选） | ⚠️ 部分实现 | 留言邮箱校验、文章内容 XSS 过滤；注册页未使用自定义正则 |
| **异步编程**（可选） | ✅ 已实现 | 所有数据库操作均使用 async/await |

## 二、未完成功能

### 1. 博文模块缺失

| 缺失功能 | 说明 | 涉及技术点 |
| :--- | :--- | :--- |
| **文章编辑页** | 只有创建，没有编辑功能，`Post.UpdatedAt` 字段从未被赋值 | 面向对象、Linq |
| **文章删除** | 没有删除页面 | 异常处理 |
| **标签系统** | 没有 Tag 模型，没有文章-标签多对多关系，没有标签筛选 UI | 面向对象、Linq |
| **Markdown 渲染** | Markdig NuGet 包未安装，文章内容以纯文本显示（仅替换换行符） | Markdig |
| **MathJax 公式渲染** | 没有在页面中引入 MathJax 脚本 | MathJax |

### 2. 相册模块缺失

| 缺失功能 | 说明 | 涉及技术点 |
| :--- | :--- | :--- |
| **瀑布流布局** | 当前使用普通 Bootstrap 网格，非 Pinterest 风格瀑布流 | CSS |
| **图片点击放大预览** | 没有 lightbox / 弹窗查看大图功能 | JavaScript |

### 3. 后台管理缺失

| 缺失功能 | 说明 | 涉及技术点 |
| :--- | :--- | :--- |
| **全局异常处理中间件** | 只有默认 `/Error` 页面，没有自定义中间件类记录异常日志 | 异常处理 |
| **操作日志** | 没有 OperationLog 模型，没有审计日志服务和界面 | 数据库、异常处理 |
| **用户管理** | 没有用户列表、角色分配等管理界面 | 面向对象 |

### 4. 其他缺失

| 缺失功能 | 说明 | 涉及技术点 |
| :--- | :--- | :--- |
| **注册页正则校验** | 注册使用数据注解 `[EmailAddress]`，未使用自定义正则验证密码强度 | 正则表达式 |
| **自定义 CSS/JS** | site.css 和 site.js 仍为模板默认内容，无样式美化 | 前端 |
